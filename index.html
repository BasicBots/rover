Features of Rover
We provide you the best features in Rover!
User Information

Rover is able to obtain user information you might not be able to view and puts it in a easy to view format. Includes, user name, id, bot, etc

Server Information

Rover is able to pull your server info and display it right in chat. Rovers instant response shows you name, members, region, etc.

Role Managment

Can't find a member in the server list to assign a role. With Rover all you need is to use our role command and let the bot give the role.
