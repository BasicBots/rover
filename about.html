About Rover.

Rover is a multipurpose utility bot for your discord server that is able to pull important and hidden info for server owners and display it with an easy to read format. Rover specializes in obtaining and displaying user information, server information, and role management. Rover is very user friendly and provides a great service for server owners and moderators.
